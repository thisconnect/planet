# Planet

Run
---

	cd bin
	./planet

- Run the tests http://127.0.0.1:8999/tests/
- Browse the examples planet/examples/index.html (Chrome does not allow Websockets from filesystem) 

Setup
-----

	git submodule update --init --recursive

Requires
--------

  - Node.js 0.6.11

#### Submodules

  - Socket.IO 0.9.5
  - Socket.IO client 0.9.5
  - Optparse-js 1.0.3
  - UglifyJS 1.2.6
  - Testigo
